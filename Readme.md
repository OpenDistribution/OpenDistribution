OpenDistribution
================

Cross-Platform Support
----------------------

![OpenDistribution running on Ubuntu](OpenDistributionOnUbuntu.png)

Running From Source
-------------------

### Windows

1. Install [node.js](https://nodejs.org/en/).
2. Run `RetreiveDependencies.bat`.
3. Run `Run.bat`.

### Ubuntu

1. Install [node.js](https://nodejs.org/en/).
2. Run `RetreiveDependencies.sh`.
3. Run `Run.sh`.
